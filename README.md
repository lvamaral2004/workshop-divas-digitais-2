# Workshop de 1 ano de Divas Digitais: Projeto Piloto

> **Base de Código para a Atividade Prática de Introdução ao Desenvolvimento Web (HTML, CSS, JS)**

---

## Visão Geral do Repositório

Este repositório serve como o **projeto piloto** (ou código base) fornecido às participantes do 2º Workshop das Divas Digitais. Ele é o ponto de partida para a atividade prática, cujo objetivo é consolidar o aprendizado dos fundamentos de HTML, CSS e JavaScript através da **CUSTOMIZAÇÃO GUIADA** de um site.

O projeto é didático e focado na prática, permitindo que as participantes apliquem gradualmente os conceitos apresentados durante o workshop.

Diferentemente da metodologia anterior, nesta atividade as participantes receberão um **template de site previamente estruturado**, porém com aparência neutra e sem interatividade. O desafio será **transformar esse site “sem vida” em uma página visualmente atrativa e interativa**, aplicando modificações em HTML, CSS e JavaScript.

O template inicial será disponibilizado **em tons de preto, branco e cinza**, contendo apenas a estrutura básica do site.

---

## Estrutura do Site

O site piloto consiste em duas páginas essenciais:

1. **Home (`index.html`):** Página inicial com a navegação principal.
2. **Sobre Mim (`sobre.html`):** Uma página secundária focada no perfil.

Cada página inclui uma estrutura padrão para praticar a arquitetura web:

* `Header` (cabeçalho)
* `Nav` (barra de navegação)
* Duas Seções de Conteúdo (combinação de texto e imagem)
* `Footer` (rodapé)

---

## Conceitos Fundamentais Abordados

O projeto integra as três pilhas básicas do desenvolvimento web, cada uma ministrada por uma instrutora, garantindo uma prática completa:

### Markup e Estrutura (HTML - Instrutora Érika)

O foco do HTML está na correta **organização e personalização do conteúdo da página**:

* **Estrutura do Documento:** Uso de `<header>`, `<main>` e `<footer>`.
* **Organização de Conteúdo:** Utilização de `<section>` e `<div>`.
* **Elementos Básicos:** Uso de `<h1>`, `<h2>`, `<p>` e `<img>`.
* **Links e Navegação:** Uso de `<nav>` e `<a>` para conectar páginas.
* **Customização:** Alteração de textos, títulos e imagens do template base.

### Estilização e Design (CSS - Instrutora Débora)

O foco do CSS está em transformar o template neutro em um site visualmente atrativo:

* **Reset Inicial:** Ajuste de margens e *paddings*.
* **Cores e Tipografia:** Alteração de cores, fontes e fundos.
* **Layout:** Organização dos elementos utilizando **Flexbox**.
* **Estilização de Componentes:** Personalização de botões, links e seções.
* **Interações Visuais:** Aplicação de efeitos **`:hover`** e **`transition`**.

### Interatividade e Comportamento (JavaScript - Instrutora Lívia)

O foco do JavaScript está em adicionar interatividade ao site:

* **Manipulação do DOM:** Alteração de conteúdo e estilos via código.
* **Eventos de Clique:** Criação de botões interativos.
* **Feedback ao Usuário:** Exibição de mensagens ou ações na página.
* **Comportamentos Simples:** Implementação de pequenas interações que tornem o site mais dinâmico.

---

## Desafio da Atividade Prática

Durante o workshop, as participantes deverão **customizar o template fornecido**, realizando modificações que tornem o site mais personalizado e visualmente interessante.

Algumas das tarefas propostas incluem:

* Alterar textos e títulos da página
* Inserir ou substituir imagens
* Modificar cores e fontes do site
* Estilizar botões e links
* Adicionar pequenas interações com JavaScript

O objetivo é permitir que as participantes **experimentem a construção de um site de forma prática e criativa**, aplicando diretamente os conceitos aprendidos.
